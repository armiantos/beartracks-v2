## Getting started

To run the full stack in development mode

1. Rename `./client/.env.sample` to `./client/.env`

2. Run the following commands in two separate terminals

```bash
cd ./client
npm start
```

```bash
cd ./server
go run .
```

Then visit `http://localhost:3000` in your browser
